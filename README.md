# Spore's Throne
##### 

## "What defines the experience?"
Spore's Throne blends the strategic depth of turn-based combat with a grim atmosphere and criativity from MOBAs. By bypassing traditional engines, the project priotiritizes custom-built system, ensuring that every mechanic is tailored to provide a better experience.

## "Why Make this Game?"
Spore's Throne is a mix of my love for dark fantasy aesthetic, game dev, and art. This project serves as a hobby lab, allowing me to dive deep into programming and game development from scratch (without a game engine).
